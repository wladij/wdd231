DevElevate - Site Plan

1️⃣ Site Name

Name: DevElevate

Reason: The name represents the goal of the site: helping developers elevate their skills with resources, courses, and challenges.

2️⃣ Site Purpose

DevElevate is an educational platform providing resources, guides, and interactive challenges to improve web development skills. The site will serve as a learning hub for beginners and growing developers.

3️⃣ Scenarios

  • Where can I find updated resources for HTML, CSS, and JavaScript?
  • What practical challenges can I do to improve my web development skills?

4️⃣ Color Scheme

Primary Color: #441752 (Purple) - Used for headings and main links.

Secondary Color: #8174A0 (Purple Gradient) - Used for buttons and highlights.

5️⃣ Typography

Primary Font: Atkinson Hyperlegible Mono (Sans-serif) - Used for headings and titles.

Secondary Font: Roboto (Sans-serif) - Used for body text.
